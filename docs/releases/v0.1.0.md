# MuseDock v0.1.0

Initial pre-release candidate for MuseDock Open, a local-first desktop AI client for user-owned OpenAI-compatible providers.

## Highlights

- ChatGPT-like local desktop client built with Tauri, React, and Rust.
- Users configure their own OpenAI-compatible provider and API key.
- API keys are stored in the operating system keychain.
- Chat supports streaming output and Markdown rendering.
- Conversations are saved locally.
- Image generation saves files locally and keeps local image history.
- Multiple provider profiles are supported.
